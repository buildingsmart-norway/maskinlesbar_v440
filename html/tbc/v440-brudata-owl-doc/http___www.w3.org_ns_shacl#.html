This vocabulary defines terms used in SHACL, the W3C Shapes Constraint Language. This vocabulary is under development by the W3C Data Shapes Working Group. The status of each term in this vocabulary is assumed to be stable unless they are marked using Term-centric Semantic Web Vocabulary Annotations.
