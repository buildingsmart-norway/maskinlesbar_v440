Property sh:flags

rdf:type
rdf:Property
rdfs:comment
This property is an optional constraint parameter that defines flags to be used with regular expression pattern matching. When present it requires the presence of sh:pattern. Its syntax is the same as that used by the SPARQL REGEX() function. The constraint is violated when any value node fails to match the regular expression defined by the pattern and modified by the flags.
